# User Stories - Sistem Manajemen Tugas (Day 2)

## Manajemen Pengguna (User Management)
- **US-001**: Sebagai pengguna baru, saya ingin bisa membuat akun dengan username, agar saya bisa menggunakan aplikasi.
- **US-002**: Sebagai pengguna, saya ingin bisa login dengan username, agar saya bisa mengakses tugas-tugas saya.
- **US-003**: Sebagai pengguna, saya ingin melihat profil saya, agar saya tahu informasi akun saya.

## Manajemen Tugas Tingkat Lanjut (Enhanced Task Management)
- **US-004**: Sebagai pengguna, saya ingin mengkategorikan tugas (Kerja/Pribadi/Belajar), agar saya bisa mengorganisir tugas dengan lebih baik.
- **US-005**: Sebagai pengguna, saya ingin menambahkan tenggat waktu (due date) pada tugas, agar saya tahu kapan batas waktunya.
- **US-006**: Sebagai pengguna, saya ingin melihat tugas yang terlambat (overdue), agar saya bisa memprioritaskan yang mendesak.
- **US-007**: Sebagai pengguna, saya ingin menugaskan (assign) tugas ke pengguna lain, agar bisa berkolaborasi.

## Penyaringan & Pengorganisasian (Filtering & Organization)
- **US-008**: Sebagai pengguna, saya ingin menyaring tugas berdasarkan kategori, agar fokus pada jenis tugas tertentu.
- **US-009**: Sebagai pengguna, saya ingin melihat tugas yang akan jatuh tempo dalam 3 hari, agar bisa bersiap.
- **US-010**: Sebagai pengguna, saya ingin melihat statistik tugas saya, agar tahu tingkat produktivitas saya.
